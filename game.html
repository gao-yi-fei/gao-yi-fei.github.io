暴塔 v0.2.2 / Web

抽象但上头的随机回合战斗

每回合玩家按行动点依次出手，技能由角色概率池随机触发。护盾会格挡一次伤害，治疗、沉默、换血、连击和大招会把局面搅得很热闹。

本地模式

对战方式

在线模式使用 WebRTC 点对点连接；房间码只用于建立连接。
本地双人可以直接在同一页面选择两名角色开打。

联机说明

一方创建房间，另一方输入房间码加入。
  1. 房主创建房间，把房间码发给对手。
  2. 双方各自选择角色与名字。
  3. 房主点击开始，之后只在轮到自己时能行动。
  4. 如果联机库加载失败，仍可使用本地双人模式。
